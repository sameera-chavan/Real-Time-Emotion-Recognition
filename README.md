Real-Time Facial Emotion Recognition:
A real-time facial emotion recognition system built using Python, TensorFlow/Keras, OpenCV, and MTCNN. The model detects faces from live webcam feed and classifies emotions into 7 categories: Happy, Sad, Neutral, Angry, Fear, Surprise, Disgust.
Trained on the FER-2013 dataset and deployed on Jetson Nano.

Features:
Real-time face detection using MTCNN
Emotion classification using a custom CNN model
Works on webcam/live camera
Optimized for edge devices (Jetson Nano)
Lightweight & fast inference

Tech Stack:
Python
TensorFlow / Keras
OpenCV
MTCNN
Numpy, Pandas
